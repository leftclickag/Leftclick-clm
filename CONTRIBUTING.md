# Contributing

Vielen Dank für Ihr Interesse an der LeftClick CLM-Plattform!

## Entwicklung

1. Forken Sie das Repository
2. Erstellen Sie einen Feature-Branch (`git checkout -b feature/AmazingFeature`)
3. Committen Sie Ihre Änderungen (`git commit -m 'Add some AmazingFeature'`)
4. Pushen Sie zum Branch (`git push origin feature/AmazingFeature`)
5. Öffnen Sie einen Pull Request

## Code-Standards

- Verwenden Sie TypeScript für alle neuen Dateien
- Folgen Sie den ESLint-Regeln
- Schreiben Sie aussagekräftige Commit-Messages
- Fügen Sie Tests hinzu, wenn möglich

## Fragen?

Bei Fragen erstellen Sie bitte ein Issue im Repository.
